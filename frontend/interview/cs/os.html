# 进程和线程

进程是并发执行的程序在执行过程中分配和管理资源的基本单位，是系统分配和调度资源的最小单位；

线程是进程中的一个控制单元，负责进程中程序的执行，是 CPU 调度的最小单位；一个进程有多个线程，多个线程可以共享进程的资源；
