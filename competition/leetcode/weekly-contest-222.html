# A. 卡车上的最大单元数 (opens new window)

# 解法一： 贪心

# 算法思想

卡车装的箱子数量和单元数没有关系，所以贪心的选择单元数多的箱子装车。

# 参考代码

class Solution {
public:
    int maximumUnits(vector<vector<int>>& a, int s) {
        sort(a.begin(), a.end(), [](const vector<int> &p, const vector<int> &q) {
            return p[1] > q[1];
        });
        int ans = 0;
        for (auto &v : a) {
            int t = min(v[0], s);
            ans += t * v[1];
            s -= t;
        }
        return ans;
    }
};

# B. 大餐计数 (opens new window)

# 解法一：枚举 + 哈希表优化

# 算法思想

使用哈希表统计不同美味程度的餐品个数，对于每种美味程度的餐品枚举 20,21,22,⋯,2212^0, 2^1, 2^2, \cdots, 2^{21}20,21,22,⋯,221，统计可能组成的组合数目。对一个组合 (x,y)(x, y)(x,y)，分两种情况：

  • 当 x=yx = yx=y 时，即从一组相同的数中选择两两组合，共 1+2+⋯+n−1=n(n−1)21 + 2 + \cdots + n - 1 = \frac{n(n - 1)}{2}1+2+⋯+n−1=2n(n−1)​ 种；
  • 当 x≠yx \ne yx=y 时，即从两组数中选择两两组合，共 m∗nm * nm∗n 种。

最后，对于每种组合都计算了两次，结果除以 222。

# 参考代码

using LL = long long;
const int MOD = 1e9 + 7;
class Solution {
public:
    int countPairs(vector<int>& ds) {
        unordered_map<int, int> mp;
        for (auto &x : ds) {
            ++mp[x];
        }
        LL ans = 0;
        for (auto [x, v] : mp) {
            for (int k = 0; k <= 21; ++k) {
                int t = (1 << k);
                if (!mp.count(x) || !mp.count(t - x)) continue;
                if (t - x == x) {
                    ans += 1LL * mp[x] * (mp[x] - 1);
                } else {
                    ans += 1LL * mp[x] * mp[t - x];
                }
            }
        }
        return ans / 2 % MOD;
    }
};
