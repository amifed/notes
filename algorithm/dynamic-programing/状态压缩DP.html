# 最短 Hamilton 路径

状态定义

f(i,j)(0≤i≤2n,0≤j<n)f(i,j)\;(0≤i≤2^n,0≤j<n)f(i,j)(0≤i≤2n,0≤j<n) 表示 "点被经过的状态" 对应的二进制数为 iii ，且目前处于点 jjj 时的最短路径。

状态计算

对于 f(i,j)f(i,j)f(i,j)，即当前时刻 "被经过的点的状态" 对应的二进制数为 iii ，处于点 jjj（此时状态 iii 的二进制位的第 jjj 位应该为 111 ，即 (i≫j)&1=1(i\gg j)\;\&\;1 = 1(i≫j)&1=1）。

因为 jjj 只能被恰好经过一次，所以一定是当前时刻经过，故上一次 "被经过的点的状态" 对应的二进制数的第 jjj 位应该为 000，即 i⊕(1≪j)=1i\oplus(1\ll j)=1i⊕(1≪j)=1。另外上一次所处的位置可能是 i⊕(1≪j)=1i\oplus(1\ll j)=1i⊕(1≪j)=1 中任意一个是 111 的数位 kkk，从 kkk 走到 jjj 需经过 weight(k,j)weight(k,j)weight(k,j) 的路程，考虑所有这样的 kkk 取最小值。

总结来说，对于当前点 jjj ，找一个状态，再找一个中间点 kkk，更新 kkk 点 到 jjj 点的最短路，该状态要满足 jjj 点的状态中 jjj 点被走过，kkk 点的状态 jjj 点没被走过、kkk 点被走过。

f(i,j)=min⁡{f(i⊕(1≪j),k)+weight(k,j)},0≤k<n,((i≫j)&1)=1f(i,j)=\min\{f(i\oplus(1 \ll j),k)+weight(k,j)\},\; 0≤k<n,\;((i \gg j) \;\&\; 1) = 1f(i,j)=min{f(i⊕(1≪j),k)+weight(k,j)},0≤k<n,((i≫j)&1)=1

初始条件

在起点时，有 f(1,0)=0f(1,0)=0f(1,0)=0 ，即只经过点 000 （ iii 只有第 000 位为 111 ），目前处于起点 000 ，最短路长度为 000 。方便起见，将 fff 其他值初始化为无穷大

最终结果

f(2n−1,n−1)f(2^n-1,n-1)f(2n−1,n−1)，即经过所有点（ iii 的所有位都是 111 ），处于终点 n−1n-1n−1 的最短路。

参考代码（C++） 
#include <iostream>
#include <cstring>
using namespace std;

const int N = 20, M = 1 << 20;

int n;
int w[N][N];
int f[M][N];

int main() {
    cin >> n;
    for (int i = 0; i < n; ++i) for (int j = 0; j < n; ++j) cin >> w[i][j];
    
    memset(f, 0x3f, sizeof f);
    f[1][0] = 0;
    for (int i = 1; i < 1 << n; ++i) {
        for (int j = 0; j < n; ++j) if (i >> j & 1) {
            for (int k = 0; k < n; ++k) if ((i ^ 1 << j) >> k & 1) 
                f[i][j] = min(f[i][j], f[i ^ 1 << j][k] + w[k][j]);
        }
    }
    cout << f[(1 << n) - 1][n - 1];
    return 0;
}
