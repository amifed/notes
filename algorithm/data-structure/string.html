# KMP 算法 nextnextnext 数组的求法

  1. 初始化 next[1]=j=0next[1]=j=0next[1]=j=0，假设 next[1∼i−1]next[1\sim i-1]next[1∼i−1] 已求出，求 next[i]next[i]next[i]。
  2. 不断尝试扩展匹配长度 jjj，如果扩展失败（下一个字符不相等），另 j=next[j]j=next[j]j=next[j]，直至 jjj 为 000（应该重新从头开始匹配）。
  3. 如果能扩展成功，匹配长度 jjj 就增加 111。next[i]=jnext[i]=jnext[i]=j。
