    # 3195 有趣的数

    我们把一个数称为有趣的，当且仅当：

    • 它的数字只包含 0,1,2,30,1,2,30,1,2,3，且这四个数字都出现过至少一次。
    • 所有的 000 都出现在所有的 111 之前，而所有的 222 都出现在所有的 333 之前。
    • 最高位数字不为 000。

    因此，符合我们定义的最小的有趣的数是 201320132013。

    除此以外，444 位的有趣的数还有两个：203120312031 和 230123012301。

    请计算恰好有 nnn 位的有趣的数的个数。

    由于答案可能非常大，只需要输出答案除以 109+7109+7109+7 的余数。

    # 输入格式

    输入只有一行，包括恰好一个正整数 nnn。

    # 输出格式

    输出只有一行，包括恰好 nnn 位的整数中有趣的数的个数除以 109+710^9+7109+7 的余数。

    # 数据范围

    4≤n≤10004≤n≤10004≤n≤1000

    # 输入样例：

    4

    # 输出样例：

    3

    # 解题思路

    根据题目关系，可以将数分为两组，(0,1)(0,1)(0,1) 和 (2,3)(2,3)(2,3)，两组之间互不影响；

    我们设 (0,1)(0,1)(0,1) 组有 kkk 位数，则 (2,3)(2,3)(2,3) 组有 n−kn-kn−k 位数，首先由于每个数字都要出现至少一次，所以 kkk 的取值范围为 2≤k≤n−22≤k≤n-22≤k≤n−2。

    对于 (0,1)(0,1)(0,1) 组的 kkk 个数，由于 000 必须在 111 前且最高位不能为 000，所以这 kkk 个数只能在后 n−1n-1n−1 位中选，故有 Cn−1kC_{n-1}^kCn−1k​。

    对于 (0,1)(0,1)(0,1) 组中的 kkk 个数，000 至少有一个，且 000 必须在 111 前，所以排列可以为 011…11,0011…11,…,00…001011\dots11,0011\dots11,\dots,00\dots001011…11,0011…11,…,00…001 共 k−1k-1k−1 种。同理 (2,3)(2,3)(2,3) 组有 n−k−1n-k-1n−k−1 种可能。

    故计算公式为 ∑2n−2Cn−1k(k−1)(n−k−1)\displaystyle\sum_2^{n-2}C_{n-1}^k(k-1)(n-k-1)2∑n−2​Cn−1k​(k−1)(n−k−1)。

    #include <iostream>
using namespace std;

const int N = 1e3 + 7;
const int MOD = 1e9 + 7;
int n;
int C[N][N];

int main() {
    cin >> n;
    for (int i = 0; i <= n; ++i) {
        for (int j = 0; j <=i; ++j) {
            if (!j) C[i][j] = 1;
            else C[i][j] = (C[i-1][j] + C[i-1][j-1]) % MOD;
        }
    }
    int ans = 0;
    for (int k = 2; k <= n - 2; ++k) {
        ans = (ans + 1LL * C[n-1][k] * (k-1) % MOD * (n-k-1)) % MOD;
    }
    cout << ans;
    return 0;
}
