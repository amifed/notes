# 递归

  • 要计算 aba^bab 时，可以先递归地计算出 t=a⌊b/2⌋t = a^{\lfloor b/2 \rfloor}t=a⌊b/2⌋；
  • 根据递归计算的结果，如果 bbb 为偶数，那么 ab=t2a^b = t^2ab=t2；如果 nnn 为奇数，那么 ab=t2aa^b = t^2aab=t2a；
  • 递归的边界为 b=0b=0b=0，a0=1a^0=1a0=1。

# 迭代

十进制整数进行二进制拆分：b=2t0+2t1+⋯+2ti+⋯+2tnb=2^{t_0}+2^{t_1}+\cdots+2^{t_i}+\cdots+2^{t_n}b=2t0​+2t1​+⋯+2ti​+⋯+2tn​，当 bbb 的二进制的 iii 位为 111 时，ti=it_i = iti​=i，否则 ti=0t_i=0ti​=0；

故，ab=a2t0+2t1+⋯+2tk=a2t0a2t1⋯a2tka^b=a^{2^{t_0}+2^{t_1}+\cdots+2^{t_k}}=a^{2^{t_0}}a^{2^{t_1}}\cdots a^{2^{t_k}}ab=a2t0​+2t1​+⋯+2tk​=a2t0​a2t1​⋯a2tk​

参考代码（C++） 
#include <iostream>
using namespace std;

int fastPow(int a, int b, int p) {
    if (b == 0) return 1 % p;
    int tmp = fastPow(a, b >> 1, p) % p;
    if (b & 1) return 1LL * tmp * tmp % p * a % p;
    return 1LL * tmp * tmp % p;
}

int fastPow(int a, int b, int p) {
    int ans = 1 % p;
    while (b) {
        if (b & 1) ans = 1LL * ans * a % p;
        a = 1LL * a * a % p;
        b >>= 1;
    }
    return ans;
}

int main() {
    int a, b, p;
    scanf("%d %d %d", &a, &b, &p);
    printf("%d", fastPow(a, b, p));
    return 0;
}

# 64 位整数乘法 (opens new window)

类似于快速幂的思想，整数 bbb 用二进制表示，即 b=ck−12k−1+ck−22k−2+⋯+c020b=c_{k-1}2^{k-1}+c_{k-2}2^{k-2}+\cdots +c_02^0b=ck−1​2k−1+ck−2​2k−2+⋯+c0​20，那么，

a∗b=ck−1∗a∗2k−1+ck−2∗a∗2k−2+⋯+c0∗a∗20a*b=c_{k-1}*a*2^{k-1}+c_{k-2}*a*2^{k-2}+\cdots +c_0*a*2^0a∗b=ck−1​∗a∗2k−1+ck−2​∗a∗2k−2+⋯+c0​∗a∗20。

当 ci=1c_i=1ci​=1 时，将乘积累加到答案中即可。

参考代码（C++） 
#include <iostream>
using namespace std;
using LL = long long;

LL mul(LL a, LL b, LL p) {
    LL ans = 0;
    while (b) {
        if (b & 1) ans = (ans + a) % p;
        a = 2 * a % p;
        b >>= 1;
    }
    return ans;
}

int main() {
    LL a, b, p;
    cin >> a >> b >> p;
    cout << mul(a, b, p);
    return 0;
}

# 最短 Hamilton 路径 (opens new window)

# 状态定义

f(i,j)(0≤i≤2n,0≤j<n)f(i,j)\;(0≤i≤2^n,0≤j<n)f(i,j)(0≤i≤2n,0≤j<n) 表示 "点被经过的状态" 对应的二进制数为 iii ，且目前处于点 jjj 时的最短路径。

# 状态计算

对于 f(i,j)f(i,j)f(i,j)，即当前时刻 "被经过的点的状态" 对应的二进制数为 iii ，处于点 jjj（此时状态 iii 的二进制位的第 jjj 位应该为 111 ，即 (i≫j)&1=1(i\gg j)\;\&\;1 = 1(i≫j)&1=1）。

因为 jjj 只能被恰好经过一次，所以一定是当前时刻经过，故上一次 "被经过的点的状态" 对应的二进制数的第 jjj 位应该为 000，即 i⊕(1≪j)=1i\oplus(1\ll j)=1i⊕(1≪j)=1。另外上一次所处的位置可能是 i⊕(1≪j)=1i\oplus(1\ll j)=1i⊕(1≪j)=1 中任意一个是 111 的数位 kkk，从 kkk 走到 jjj 需经过 weight(k,j)weight(k,j)weight(k,j) 的路程，考虑所有这样的 kkk 取最小值。

总结来说，对于当前点 jjj ，找一个状态，再找一个中间点 kkk，更新 kkk 点 到 jjj 点的最短路，该状态要满足 jjj 点的状态中 jjj 点被走过，kkk 点的状态 jjj 点没被走过、kkk 点被走过。

f(i,j)=min⁡{f(i⊕(1≪j),k)+weight(k,j)},0≤k<n,((i≫j)&1)=1f(i,j)=\min\{f(i\oplus(1 \ll j),k)+weight(k,j)\},\; 0≤k<n,\;((i \gg j) \;\&\; 1) = 1f(i,j)=min{f(i⊕(1≪j),k)+weight(k,j)},0≤k<n,((i≫j)&1)=1

# 初始条件

在起点时，有 f(1,0)=0f(1,0)=0f(1,0)=0 ，即只经过点 000 （ iii 只有第 000 位为 111 ），目前处于起点 000 ，最短路长度为 000 。方便起见，将 fff 其他值初始化为无穷大

# 最终结果

f(2n−1,n−1)f(2^n-1,n-1)f(2n−1,n−1)，即经过所有点（ iii 的所有位都是 111 ），处于终点 n−1n-1n−1 的最短路。

参考代码（C++） 
#include <iostream>
#include <cstring>
using namespace std;

const int N = 20, M = 1 << 20;

int n;
int w[N][N];
int f[M][N];

int main() {
    cin >> n;
    for (int i = 0; i < n; ++i) for (int j = 0; j < n; ++j) cin >> w[i][j];
    
    memset(f, 0x3f, sizeof f);
    f[1][0] = 0;
    for (int i = 1; i < 1 << n; ++i) {
        for (int j = 0; j < n; ++j) if (i >> j & 1) {
            for (int k = 0; k < n; ++k) if ((i ^ 1 << j) >> k & 1) 
                f[i][j] = min(f[i][j], f[i ^ 1 << j][k] + w[k][j]);
        }
    }
    cout << f[(1 << n) - 1][n - 1];
    return 0;
}

# 起床困难综合症 (opens new window)

​本题意为选择 [0,m][0,m][0,m] 之间的一个整数 x0x_0x0​，经过给定的 nnn 次位运算，使结果 ansansans 最大。

位运算的主要特点之一是在二进制表示下不进位。在 x0x_0x0​ 可以任意选择的情况下，参与位运算的各个位（bit）之间是独立无关的。换言之，对于任意的 k(0≤k<30)k\;(0≤k<30)k(0≤k<30)，“ansansans 的第 kkk 位是几“ 只与 “x0x_0x0​ 的第 kkk 位是几” 有关，与其他位无关。所以从高位到低位，依次考虑 x0x_0x0​ 的每一位填 000 还是填 111。

x0x_0x0​ 的第 kkk 位应该填 111 ，当且仅当同时满足两个条件：

  1. 已经填好的更高位构成的数加上 1≪k1\ll k1≪k 后不超过 mmm；
  2. 用每个参数的第 kkk 位参与位运算。若初值为 111 ，则 nnn 次位运算后结果为 111；若初值为 000 ，则 nnn 次位运算后结果为 000；

如果不满足上述条件，要么填 111 会超过 mmm 的范围，要么填 111 不如填 000 更优，因为如果填 111 ，而 nnn 次位运算后变为 000，则增加了 x0x_0x0​，却没增大ans，或者低位填 111 导致 x0x_0x0​ 可能大于 mmm 而得不到最优解。确定 x0x_0x0​ 的每一位后，自然得到 ansansans 的值。

参考代码（C++） 
#include <iostream>
#define x first
#define y second
using namespace std;

const int N = 1e5 + 7;

int n, m;
pair<string, int> ops[N];

int cacl(int bit, int now) {
    for (int i = 1; i <= n; ++i) {
        int x = ops[i].y >> bit & 1;
        if (ops[i].x == "AND") now &= x;
        else if (ops[i].x == "OR") now |= x;
        else now ^= x;
    }
    return now;
}

int main () {
    cin >> n >> m;
    for (int i = 1; i <= n; ++i) cin >> ops[i].x >> ops[i].y;
    int val = 0, ans = 0;
    for (int bit = 29; bit >= 0; --bit) {
        int ret0 = cacl(bit, 0);
        int ret1 = cacl(bit, 1);
        if (val + (1 << bit) <= m && ret0 < ret1) 
            val += 1 << bit, ans ^= ret1 << bit;
        else ans ^= ret0 << bit;
    }
    cout << ans;
    return 0;
}
