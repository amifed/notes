# CSS

## 定位

相对定位允许相对于其正常起点调整元素的位置

绝对定位允许相对于其第一个非静态定位的祖先元素调整位置

相对定位和绝对定位的项目都不会影响周围的静态和固定元素（绝对定位将从流中移除，相对定位的项目占据其原始位置）
