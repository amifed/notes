# 贪心

贪心是一种在每次决策是采取当前意义下最优策略的算法，使用贪心算法要求问题的整体最优性可以由局部最优性导出。

常见的证明方法：

1. 微扰（邻项交换）
2. 范围缩放
3. 决策包容性
4. 反证法
5. 数学归纳法

## [防晒](https://www.acwing.com/problem/content/description/112/)

按照 `minSPF` 递减的顺序排序，依次考虑每头奶牛

对于每头奶牛，扫描一边所有的防嗮霜，在这头奶牛能用的防晒霜里找 `SPF` 值最大的使用
