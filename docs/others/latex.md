# Latex 常用符号

使用 `$` 符号包裹

## 空格

| 名称       | 代码        | 展示        |
| ---------- | ----------- | ----------- |
| qquad 空格 | `a\qquad b` | $a\qquad b$ |
| quad 空格  | `a \quad b` | $a \quad b$ |
| 大空格     | `a\ b`      | $a\ b$      |
| 中空格     | `a\;b`      | $a\;b$      |
| 小空格     | `a\,b`      | $a\,b$      |
| 无空格     | `ab`        | $ab$        |
| 紧贴       | `a\!b`      | $a\!b$      |

## 分数

`\farc{a}{b}` ，$\frac{a}{b}$

`\displaystyle\frac{a}{b}`，$\displaystyle\frac{a}{b}$

## 符号

`\in`，$\in$

`\ni`，$\ni$

`\le`，$\le$

## 上下标

`A_n^m`，$A_n^m$

## 省略号

`\dots`、`\ldots`，$a,\dots,b$

`\cdots`，$a,\cdots,b$

`\vdots`，$\vdots$
